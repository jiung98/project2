About Us

JDT 농수산물 데이터 제공 서비스는 다양한 데이터를 통해 고객이 전략적 결정을 내리는 데 도움을 드립니다.

주요 서비스

데이터 개방

농수산물 데이터를 쉽고 빠르게 검색할 수 있는 데이터 개방 서비스를 제공합니다.

데이터 분석 서비스

농수산물 시장 분석을 위한 맞춤형 데이터 분석 서비스를 제공합니다.

데이터 시각화

데이터 시각화 도구를 통해 데이터를 더 쉽게 이해할 수 있도록 도와드립니다.
